# Café Reservation and Digital Menu System

## Description
This project is a web-based Café Reservation and Digital Menu System developed using PHP and MySQL. It allows customers to view the menu digitally, book tables, and place orders online. The system also provides an admin panel to manage menu items, reservations, and orders.

---

## Features

### 👤 User Module
- View digital menu (food & beverages)
- Register and login
- Book table reservations
- Place food orders
- View order details

### 📱 Digital Menu System
- Displays menu items with price
- Categorized items (Drinks, Snacks, Meals)
- Easy navigation for users
- Dynamic updates from database

### 🪑 Reservation System
- Book tables online
- Select date and time
- Store reservation details in database

### 🛠 Admin Module
- Admin login
- Add / update / delete menu items
- Manage customer orders
- Manage reservations
- View feedback

---

## Technologies Used
- PHP (Backend)
- MySQL (Database)
- HTML, CSS (Frontend)
